`AOP`

## 类型

1. 编译期 - 注解处理器

2. 编译期 - 字节码技术 - Javassist

3. 编译期 - 字节码技术 - ASM

4. 运行期 - 动态代理

5. 运行期 - 反射

6. AspectJ（利用 transform）

## Java 代码的三个时期（RetentionPolicy）

1. 源码期（SOURCE）

2. 编译期（CLASS）

3. 运行时（RUNTIME）

## Android 代码的四个时期（RetentionPolicy）

1. 源码期（SOURCE）

2. .class文件

3. .dex文件

4. 运行时（RUNTIME）
